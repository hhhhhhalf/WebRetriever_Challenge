赛事背景 Background

当 AI 智能体走进真实浏览器，它能否像人一样，在充满动态变化的互联网环境中独立完成任务？这是 Web Agent 从实验室走向部署的核心瓶颈。

明略科技构建了大规模 Web Agent 综合评测基准 WebRetriever，论文已被 ECCV 2026 接收。挑战赛诚邀全球研究者与开发者，共同推动从"可演示"到"能交付"的关键突破。

When AI agents enter real browsers, can they truly complete tasks independently on the open internet? Mininglamp Technology built WebRetriever, a large-scale Web Agent benchmark accepted by ECCV 2026, covering 800 live websites and 1,550 tasks across 8 domains. The NavEval framework achieves 91.2% agreement with human experts. Even the best model achieves only ~20% end-to-end task completion. The WebRetriever Challenge invites teams worldwide to validate Web Agent capabilities in the most realistic environment.

赛题说明 Task

聚焦 Protocol III（端到端任务协议）——智能体不仅导航到正确页面，更需从文本、文档、图表等多模态内容中精准抽取目标信息。所有任务遵循三大原则：权威性、交互必要性、确定性。

参赛 Agent 需要

  • 接收自然语言任务指令
  • 自主规划并执行完整操作链
  • 提取并返回任务要求的结果

赛题特征

  • 端到端评测，覆盖多行业真实网站
  • 动态页面 + 多步推理 + 跨页面聚合

详细规格见备赛期文档

The challenge centers on Protocol III (End-to-End Task Protocol) — agents must navigate to correct pages and extract target information from multimodal content. Agents receive a natural language instruction, autonomously execute full action pipelines on live websites, and return the required results. Detailed specifications will be released during the preparation period.

评测方式 Evaluation

  • 自动化评测 — 通过赛事 Bot 对话发起提测，评测完成后获得评分反馈
  • 多次提交 — 窗口期内允许多次提交，取最优成绩
  • OpenAI 兼容接口 — Agent 接入统一评测环境，团队自行管理推理资源
  • 评测技术细则将在备赛期公布

Teams initiate evaluation through the Competition Bot and receive scoring feedback. Multiple submissions allowed; best score counts. Agents connect via OpenAI-compatible APIs; teams manage their own inference resources. Detailed specs released during the preparation period.
