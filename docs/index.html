赛事背景

Background

当 AI 智能体走进真实浏览器，它能否像人一样，在充满动态变化的互联网环境中独立完成任务？这是 Web Agent 从实验室走向部署的核心瓶颈。

明略科技构建了大规模 Web Agent 综合评测基准 WebRetriever，论文已被 ECCV 2026 接收。挑战赛诚邀全球研究者与开发者，共同推动从"可演示"到"能交付"的关键突破。

When AI agents enter real browsers, can they truly complete tasks independently on the open internet — just as humans do? This is the core bottleneck for deploying Web Agents from labs to real-world scenarios.

Mininglamp Technology built WebRetriever, a large-scale Web Agent evaluation benchmark accepted by ECCV 2026. The challenge invites researchers, developers, and teams worldwide to drive the leap from "demo-ready" to "delivery-ready."

赛题说明

Task Description

聚焦 Protocol III（端到端任务协议）——智能体不仅导航到正确页面，更需从文本、文档、图表等多模态内容中精准抽取目标信息。所有任务遵循三大原则：权威性、交互必要性、确定性。

The challenge centers on Protocol III (End-to-End Task Protocol) — agents must not only navigate to the correct page, but also precisely extract target information from text, documents, charts, and other multimodal content. All tasks follow three principles: authority, mandatory interaction, and determinism.

参赛 Agent 需要

  • 接收自然语言任务指令
  • 自主规划并执行完整操作链
  • 提取并返回任务要求的结果

赛题特征

  • 端到端评测，覆盖多行业真实网站
  • 动态页面 + 多步推理 + 跨页面聚合

详细规格见备赛期文档

Agent Requirements

  • Receive a natural language task instruction
  • Autonomously plan and execute full action pipelines
  • Extract and return the required results

Task Characteristics

  • End-to-end evaluation across real websites
  • Dynamic pages + multi-step reasoning + cross-page aggregation

Detailed specs released during preparation period

评测方式

Evaluation

  • 自动化评测 — 通过赛事 Bot 对话发起提测，完成后获得评分反馈
  • 多次提交 — 窗口期内允许多次提交，取最优成绩
  • OpenAI 兼容接口 — Agent 接入统一评测环境，团队自行管理推理资源
  • 评测技术细则将在备赛期公布
  • Automated Evaluation — Initiate via Competition Bot; receive scoring feedback upon completion
  • Multiple Submissions — Submit multiple times within each window; best score counts
  • OpenAI-Compatible API — Agents connect to the unified evaluation environment; teams manage their own inference resources
  • Detailed evaluation specs will be released during the preparation period
